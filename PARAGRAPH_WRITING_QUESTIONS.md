# Paragraph Writing Questions for Logical Thinking Assessment

## Overview
The quiz system now includes paragraph writing questions (minimum 200 words) to assess students' logical thinking, critical analysis, and written communication skills alongside traditional multiple-choice questions.

## Features

### 1. Mixed Question Types
- **MCQ Questions**: Traditional multiple-choice questions for quick knowledge assessment
- **Paragraph Questions**: Long-form writing questions requiring logical reasoning and structured arguments

### 2. Question Structure

#### Paragraph Question Format
```typescript
{
  q: "Main question/topic",
  prompt: "Detailed guidelines and points to address",
  concept: "Skill being assessed",
  type: "paragraph",
  minWords: 200
}
```

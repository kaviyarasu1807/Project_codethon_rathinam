## 🎯 How to Solve a Problem

### Step 1: Select a Problem
- Click on any problem in the left panel
- Problem details load in the middle panel
- Starter code appears in the editor

### Step 2: Choose Language
- Click the language dropdown (top of editor)
- Select: JavaScript, Python, Java, or C++
- Code automatically switches to selected language

### Step 3: Write Your Solution
- Type your code in the editor
- Use the starter code as a template
- Follow the function signature provided

### Step 4: Test Your Code
- Click **"Run Code"** (green button)
- View output in the output panel
- Check test case results

### Step 5: Submit Solution
- Click **"Submit"** (blue button)
- All test cases run (including hidden ones)
- Earn points if all tests pass!

## 💡 Features Explained

### Problem List (Left Panel)
- **Title**: Problem name
- **Difficulty Badge**: 
  - 🟢 Green = Easy
  - 🟡 Amber = Medium
  - 🔴 Red = Hard
- **Category**: Topic (Arrays, Strings, etc.)
- **Points**: Score value
- **✓ Checkmark**: Already solved

### Problem Description (Middle Panel)
- **Full Description**: What you need to solve
- **Test Cases**: Example inputs and outputs
- **Hints**: Click "Show Hints" for help
- **Time Limit**: Suggested completion time

### Code Editor (Right Panel)
- **Language Selector**: Choose your language
- **Code Area**: Write your solution
- **Copy Button**: Copy code to clipboard
- **Reset Button**: Reset to starter code
- **Run Code**: Test your solution
- **Submit**: Submit for grading

### Output Panel (Bottom Right)
- **Console Output**: Execution results
- **Test Results**: Pass/fail for each test
- **Error Messages**: If code fails

### Header Bar (Top)
- **Back Button**: Return to dashboard
- **Score**: Total points earned
- **Timer**: Time spent coding
- **Progress**: Problems solved / total

## 🎮 Tips & Tricks

### Writing Code
1. **Read Carefully**: Understand the problem first
2. **Check Examples**: Look at test cases
3. **Start Simple**: Get basic solution working
4. **Test Often**: Run code frequently
5. **Use Hints**: Don't struggle too long

### Using the Editor
- **Tab Key**: Indent code (2 spaces)
- **Copy/Paste**: Use Ctrl+C / Ctrl+V
- **Reset**: Start over if stuck
- **Language Switch**: Try different languages

### Solving Problems
1. **Easy First**: Build confidence
2. **Read Hints**: After trying yourself
3. **Test Cases**: Make sure all pass
4. **Hidden Tests**: Some tests are hidden
5. **No Penalty**: Failed attempts don't hurt

### Scoring
- **First Solve**: Earn full points
- **Re-solve**: No additional points
- **Hints**: Don't affect score (in demo)
- **Time**: Doesn't affect score (in demo)

## 📊 Problem Difficulty Guide

### Easy Problems (5-10 points)
- **Time**: 3-5 minutes
- **Concepts**: Basic loops, arrays, strings
- **Examples**: Two Sum, Reverse String
- **Good For**: Beginners, warm-up

### Medium Problems (15-20 points)
- **Time**: 10-15 minutes
- **Concepts**: Data structures, algorithms
- **Examples**: Valid Parentheses, Binary Search
- **Good For**: Intermediate practice

### Hard Problems (25-30 points)
- **Time**: 20-30 minutes
- **Concepts**: Advanced algorithms, optimization
- **Examples**: Merge K Lists, Median of Arrays
- **Good For**: Advanced challenges

## 🔧 Troubleshooting

### Code Won't Run
- **Check Syntax**: Look for typos
- **Function Name**: Must match starter code
- **Return Value**: Make sure you return something
- **Language**: Verify correct language selected

### Tests Failing
- **Read Error**: Check what went wrong
- **Check Logic**: Review your algorithm
- **Edge Cases**: Consider special inputs
- **Test Cases**: Compare with examples

### Output Not Showing
- **Wait**: Give it a moment to execute
- **Refresh**: Reload the page if stuck
- **Console**: Check browser console for errors

### Can't Submit
- **Run First**: Test your code first
- **Fix Errors**: Resolve any issues
- **Wait**: Don't click multiple times
